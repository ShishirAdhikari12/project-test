# 📝 Medium Clone (Laravel)

A feature-rich **Medium-like blogging platform** built using **Laravel**. This project focuses on backend functionality and core social features such as posts, claps, and user interactions.

---

## 🚀 Features

### 👤 Authentication

- User authentication using **Laravel Breeze**
- Register, login, logout

### ✍️ Posts

- Create blog posts with:
    - Title
    - Content
    - Featured image
    - Category
    - Scheduled publish date ⏳

- Edit and delete your own posts

### 👏 Engagement

- Clap (like) posts
- View clap counts

### 👥 Social Features

- Follow / unfollow users
- Personalized feed based on users you follow

### 🧑 Profile Management

- Update:
    - Profile picture 🖼️
    - Username
    - Bio

### 🏷️ Categories

- Organize posts by category

### ⚙️ Additional

- Uses **Spatie package(s)** for enhanced functionality

---

## 🛠️ Tech Stack

- **Backend:** Laravel
- **Authentication:** Laravel Breeze
- **Database:** MySQL (or any Laravel-supported DB)
- **Packages:** Spatie
